is-grid
All grid and column classes are meant to be used as a combo class. This is to allow customization per element as needed and not cause global accidental overrides. Hence all classes start with is-.
There would be occassional cases where you may use it as a direct class, but keep it to a minimum

is-grid-column-custom

This is for custom grid columns, but prefer to directly add grid-column property on the main element class instead of adding this combo class. This is only to be used occasionally, for instance - if multiple elements with half/third/quarter combo classes need to be changed to another grid column setup.

is-grid-full-width
Grid class option that stretches full width. To be used at the section level without including page padding and container wrappers within it. It has named grid areas (`full-width` and `container`) to stretch contents or limit them. Use `-start` and `-end` with grid name areas to create dynamic column compositions.
